CSS

Linguagem de estilização e definição de comportamento sobre o HTML

Pode ser incorporada em três locais diferentes

CSS Externo
CSS externo é definido utilizando a tag link no inicio do documento HTML na seção de HEAD
< link rel="stylesheet" href="style.css" >
CSS Incorporado
Utilizando a tag Style na HEAD do HTML
CSS Inline
Definido no proprio atributo HTML pela tag Style

Model box

O HTML e CSS possuem um conceito central que é o modelo em caixa, um elemento é definido dentro de outro tendo uma estrutura comum a todos elementos HTML
