O que é JavaScript?

A. A motivação para a criação do JavaScript foi a necessidade de dinamismo e interatividade nas paginas web
B. No navegador os codigos são executados pelo computador do usuario, enquanto no servidor os codigos são executados no servidor que hospeda a aplicação, escondendo assim o codigo do acesso do cliente
C. E um ambiente controlado para executar plugin com segurança
D. O document é o objeto que representa o arquivo html, o console é a parte que permite fazer modificações temporarias e resultados na propria web, alert são mensagem que aparecem na tela para o usuario bloqueando a pagina ate a sua confirmação
