Linha do Tempo

Surgimento das redes de computadores

Durante as décadas de 1950 e 1960 surgiram pesquisas para permitir que computadores compartilhassem informações entre si.

ARPANET

Em 1969, surgiu a ARPANET, considerada a primeira grande rede de computadores.

TCP/IP

Em 1983, o protocolo TCP/IP passou a ser utilizado oficialmente, permitindo a comunicação entre diferentes redes.

DNS

O DNS surgiu para substituir a necessidade de decorar endereços numéricos (IP), permitindo utilizar nomes como www.exemplo.com.

World Wide Web

Em 1989, Tim Berners-Lee, pesquisador do CERN, criou a World Wide Web, revolucionando a forma como informações eram compartilhadas.

HTML

O HTML foi criado para estruturar documentos da Web utilizando títulos, textos, imagens e links.

HTTP

O protocolo HTTP tornou possível a comunicação entre navegadores e servidores Web.

Navegadores

Os navegadores permitem acessar páginas da Internet. Atualmente existem diversos navegadores modernos que interpretam documentos HTML.

Servidor Web

O servidor Web é responsável por armazenar e disponibilizar páginas para os usuários através da Internet.

Principais Conceitos

Internet
Rede mundial de computadores.
Web
Serviço que utiliza a Internet para disponibilizar páginas.
HTML
Linguagem de marcação utilizada na construção das páginas.
HTTP
Protocolo responsável pela comunicação entre navegador e servidor.
